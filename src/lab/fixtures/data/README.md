# Lab Fixtures

These are small raw x86 byte fixtures for the metrics command.

| Fixture | Measures |
|---|---|
| `branch_countdown` | command smoke fixture with a short branch loop |
| `register_arithmetic_loop` | register ALU and conditional branch dispatch |
| `branch_countdown_loop` | repeated backward branch and decoded-block cache reuse |
| `memory_load_store_loop` | guest-memory load/store dispatch in a loop |
| `memory_alu_loop` | memory-source ALU dispatch in a loop |
